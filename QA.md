## Additional checks performed

- **CORS:** verified live on `:8124` — request with no Origin → 200; `Origin: null` → 200 with `Access-Control-Allow-Origin: null`; `Origin: https://evil.example.com` → 403 `{"error":"Origin not allowed"}`.
- **Syntax:** `node --check` passes on every JS file in `app/` and the modified server files.
- **Loopback bind:** server logs "(loopback only)"; a non-loopback `HOST` triggers a prominent multi-line warning (code-reviewed; default unchanged).
- **Electron smoke test (Windows):** `npx electron .` launched the main process, created the mini window, and spawned the server child. Note: a previously running dev server occupied :8123 (`EADDRINUSE` in the child) — the app continues and talks to whichever server owns the configured port, which is acceptable dev behavior.
- **Reduced motion:** the 600ms chart draw-in and skeleton pulse are both disabled under `prefers-reduced-motion: reduce` (the only animations in the product).
